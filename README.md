# Analiza rynku telefonów - MobileTech
###### ML - Sieć neuronowa 

Do tego projektu wstępem jest krótka historyjka:

*Jesteś pracownikiem firmy MobileTech, która jest znana na całym świecie z międzyplatformowych aplikacji dla osób niewidomych i niedowidzących. Firma planuje stworzenie dedykowanego smartfona, ponieważ obecne modele AI nie współgrają najlepiej z ograniczeniami sprzętowymi urządzeń mobilnych.*

*Twoim zadaniem jest przeprowadzenie analizy rynku w celu ustalenia optymalnej ceny nowego urządzenia.*

*W ramach zadania powierzono Ci specyfikacje techniczne konkurencyjnych telefonów oraz ich orientacyjną cenę. Jeżeli będziesz potrafił ustalić ich cenę, to z pewnością poradzisz sobie z własnymi telefonami.*
